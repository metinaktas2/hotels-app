# 🏨 Hotels - Hotel Listing & Management App

A modern and fully responsive **hotel listing web application** built with React and Vite.  
Users can explore hotels, filter them based on various criteria, and add new hotels through a dedicated form page.

---

## 🚀 Features

### 🔍 Hotel Listing
- 🏨 Display a list of hotels
- 📱 Fully responsive design (mobile / tablet / desktop)

---

### 🎯 Filtering System
- ⭐ Filter by rating
- 💰 Filter by price
- 📍 Filter by location
- 🔎 Search by hotel name

---

### ➕ Add New Hotel
- 📝 Create new hotel entries
- Fields include:
  - Hotel name
  - Location
  - Rating
  - Price
- ✅ Form validation with error handling

---

## 🧰 Tech Stack

### ⚙️ Frontend (React / Vite)

#### 📦 Dependencies
- ⚛️ React, React DOM → UI
- 🌐 React Router DOM → Routing
- 🔄 React Query → Server state & caching
- 🔗 Axios → HTTP requests
- 🧾 Formik → Form management
- ✅ Yup → Validation
- 🎨 Tailwind CSS → Styling
- 🎯 Lucide React → Icons
- 🔔 React Toastify → Notifications

#### 🛠️ DevDependencies
- ⚡ Vite → Build tool & dev server
- 🔧 TypeScript → Type safety
- 🧹 ESLint → Code quality
